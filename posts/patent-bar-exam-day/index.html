考试时间管理策略 Link to this section #

  • 一定要用三套 PLI 模拟题来估算并练习你的速度。我的数据是：

    • 模拟题：90 分钟做完 50 题，正确率 85+%
    • 单项练习：平均每题 ≤ 3 分钟

  • 真实考试当天：前两小时只做了 40+ 题， 剩下 6–8 题明确需要大量 look up 才能确定。完成 incomplete + review marked 后只剩 1 分钟。

  • Reddit 的主流建议是：第一遍跳过难题，不要卡住。我完全同意这一点：很多人没过就是卡死在某一两道题上，导致后面的简单题没时间做。

  • 但我没有采用 Reddit 推荐的“快速过题法”。 Reddit 推荐：

    1. 第一遍先做最简单、不用 look up 的（在 1 小时内完成）
    2. 第二遍做 incomplete
    3. 第三遍做 marked + double check（在 2 - 2.5 小时内完成）
    4. 如果还有时间再扫一遍全部

我的实际考试策略 Link to this section #

  • 第一遍：尽可能做完所有题
    • 无论题目长短、是否需要 look up，我都会尝试把题做完
    • 除非完全找不到方向，至少排除 2 - 3 个错误选项
    • 如果已找到答案但没时间验证所有选项，就 mark 住
    • 第一遍基本覆盖全部题目（尽最大可能拿分）
  • 第二遍：解决 incomplete 题（ 6 - 8 道题）
    • 集中精力处理卡住的问题，避免它们拖到最后
  • 第三遍：确认第一遍中 mark 的题 （大约 10 题）
    • 重点是排除剩余不确定的选项、彻底锁定答案

为什么不用 Reddit 的快速过题法？

  • 我所有 mock exam 都是一遍做到底， 在实际考试突然换策略风险太高。
  • 我对自己的 MPEP look up 能力非常有信心（这是个巨大的前提条件）
    • 考试全程没用 subject matter index
    • 大多数题能在 3 分钟内查到所有选项对应的 MPEP 原文（不仅仅是正确答案）
  • Reread 所有题目会浪费大量时间。 不如第一遍认真做，确保能拿到能拿的分。Reddit 建议的第一遍⌜不 look up MPEP⌟我不太推荐，确保拿分最可靠的方法永远是找到 MPEP 原文。

时间管理策略不是学别人怎么做，而是在真实考试前知道自己能在多长时间内解决多少题。

Prometric Test Drive Link to this section #

在正式去考 Patent Bar 之前，我从 Reddit 上了解到 Prometric 有模拟考试流程的 test drive。因为我之前从来没有在 Prometric 考试，也不熟悉他们的安检、流程和环境，加上想提前把焦虑降到最低，于是我在考试前三周预约了一次 test drive。这 30 分钟的 test drive 对我来说非常值得。第一次过去我甚至找错了楼，但也正因为这次体验，让我在正式考试那天不会再因为找路浪费精力。考场工作人员都非常友好，我把所有想问的问题都提前问清楚了，比如虽然考试时间写 8 点开始，但如果你提早到，7:15 左右其实就可以开始。考试提供红色草稿纸和两根铅笔。如果你对考试流程有焦虑、担心突发情况，Test Drive 我会非常推荐，提前降低不确定性，会让正式考试当天轻松很多。

需要注意的是：Test Drive 用的是 Prometric 自己的绿色界面系统，不是 Patent Bar 的蓝色界面，因此无法体验真正的 PDF viewer。但真正的考试界面可以在这里预览 Test Taker US Patent Registration Examination。

考试当天流程与实际体验 Link to this section #

Patent Bar 理论上早上 8 点开考，上午三小时，午休最长一小时（可以提前结束），下午再三小时，最后是调查问卷。考完会立即在屏幕上显示 preliminary 成绩，出考场后工作人员会打印正式的 preliminary result。

关于入场与规则：

  • 只能带一瓶透明水瓶（需撕掉标签），可以自带耳塞
  • 安检较严格：掏口袋、露脚踝、金属扫描
  • 中途可以上厕所，但计时不停
  • 从卫生间回来需要重新安检，很浪费时间，因此建议尽量避免频繁离席

虽然 3 小时听起来很长，但真的考起来时间过得非常快。

我的考试补给与休息安排 Link to this section #

提前准备考试备考包：两瓶撕掉包装的水，驾照，外套，耳塞，当天早上烤好的 bagel，Barbell 蛋白棒，一杯抹茶拿铁

我平常不吃午饭，所以中间只需要少量能量补充就够支撑下午考试。我午休直接走到车里吃东西，然后在楼下散步 20 分钟，呼吸新鲜空气，把紧张情绪释放掉。回到考场后还有 10 分钟倒计时，我会深呼吸三次再进入下午场。

上午 vs 下午难度：我的主观感受 Link to this section #

我看到网上许多人说上午和下午难度会不同，但对我个人来说两场难度差不多。差别只在于：

  • 上午场：需要一点时间适应 PDF viewer 和考试节奏，搜索速度也比较慢
  • 下午场：完全上手后状态很好，节奏顺畅，做题速度稳定

下午 clear all marked questions 之后做其实已经挺有信心了，因为我每道题都查到了 MPEP 原文。（如果这样都不过，那我真的要怀疑世界了！）

龟速电脑和马赛克画质 Link to this section #

虽然很多人抱怨电脑很卡，但总体来说我没有遇到完全崩溃过的极限情况。每道题翻页大概会有一两秒迟钝，打开 MPEP 和切换到不同小节基本上没有卡顿，搜索的时候遇到比较长的篇幅会有两三秒的延迟。比较要命的是 PDF 很模糊，需要不停地用鼠标放大缩小来看内容，这里比较费时间。而且我会移动 PDF viewer 好让 MPEP 跟题目答案放在一起进行比较勾选。

考场噪音与环境 Link to this section #

考场总体安静，但我依然强烈推荐带耳塞。他们提供的降噪耳机但重量有点压头，我更喜欢用自己的耳塞。
