考试并不简单捏。

美国专利代理人资格考试（USPTO Registration Exam，也常被称为“Patent Bar”）是美国专利从业人员（专利代理人／专利律师）进入专业队列的一道关键门槛。其报考条件中，不要求必须具备法学学历，但要求申请人具备一定的 STEM（科学、技术、工程、数学）学历背景。通过该考试并完成注册手续后，考生即可获得代表发明人或专利申请人在 USPTO 面前进行专利申请审查流程（prosecution）的资格：无论是作为专利代理人（非律师）还是作为具备律师资格的专利律师，均可在专利事务中代表客户办理专利申请、答复审查意见、推动专利授权。从历年统计数据来看，该考试的通过率一直低于 50%，所以这不是一个很容易能通过的考试。本文将介绍 Patent Bar 报名考试流程，从申请到考试流程，到最终获得 registration number。

申请考试资格 Link to this section #

官方指导请参考网页 Becoming a Patent Practitioner。简单来说，需要在https://oedci.uspto.gov/OEDCI/注册账号并递交考试资格申请。关于具体每一步申请填写细节，请参考 General Requirements Bulletin 和 US Patent Agent 美國專利代理人考試—圖文申請流程及考試心得。 其中需要注意两个要求：专业学历和在美移民身份。

专业学历要求 Link to this section #

Bulletin 讲学历要求分为 A，B，C，和 D 大类，对应有不同的要求。通常来说大部分人落入 A 类。

  • 如果你有 STEM 相关的学历（本科，硕士或博士） 能落入 A 类，则只需要提由学校官方递送的成绩单（ official transcript）即可。例如我的学历是 PhD in Chemistry， 我需要向学校提交申请官方成绩单直接递送给 USPTO’s Office of Enrollment and Discipline (OED) 即可满足学历要求。
  • 如果你的学历不在 A 类，则可以尝试通过 B 类利用等价课程来满足学历要求。比如学历是与 phamarcy 相关，不满足 A 类，但可以通过 biology 或 chemistry 课程学分来达到要求。
  • C 类则是需要通过美国工程基础考试（Fundamentals of Engineering test）。
  • D 类只适用于 design patent practitioner。

移民身份 Link to this section #

国籍／移民身份要求的核心原则:USPTO 要求 注册为专利代理人或律师（practitioner） 需要满足“美国公民或永久居民（绿卡）”条件。但这不影响非移民身份（如 H-1B、F-1 OPT、CPT 等）报考考试本身。非移民身份通过考试后，通常属于 limited recognition（有限代理权限）。

非移民身份如何获得考试资格（以我本人 H-1B 为例） Link to this section #

  • 关键不是身份类别，而是 USCIS 文件中是否能证明你在美国有合法、与专利代理相关的工作权限。
  • 对于 H-1B 身份而言，最重要的是：
    • 雇主在 H-1B petition 文件（job description、petition letter 等）中明确写明：Represent patent applicants before the USPTO.
    • 在 USPTO 的资格申请页面上传完整的 H-1B petition package（包括所有 supporting documents）。
    • 我刚好遇上 H-1B extension，可以在新增 petition letter 中加入相关职责说明。若不是 extension 年份，则可能需要重新提交 H-1B amendment 才能满足 USPTO 要求。

####其他非移民身份（F-1 OPT、CPT 等）的可行路径

根据社区经验（包括小红书、论坛等分享）：只要雇主愿意提供书面说明，证明你的工作职责需要与 USPTO 交互或从事专利相关任务，F-1 OPT / CPT 也可以成功获得考试资格。

官方审批流程与时间线 Link to this section #

  • 申请资格 + 学校寄官方成绩单 → 大约 2 - 4 周会收到 USPTO OED 的决定。
  • 若获批：会收到考试资格信（authorization letter），需在 6 个月内完成考试。
  • 若资料不足：OED 会明确告知 补件要求（request for additional evidence）。

Prometric 考试 Link to this section #

在上文提及资格获批的信件中会包含一个 OED ID number，你需要这个信息在 Prometric 中心预定考位。个人建议是根据当前备考的状态提前 4 - 6 周预定考位。可以直到在考试 6 天前花少量手续费改期考试，5 天以内改期考位费不退。

Patent Bar 考试机制 Link to this section #

考试形式

  • 共 100 道选择题，分成上午与下午两场，每场 50 题 / 3 小时。
  • 每题有 5 个选项。
  • 其中 10 题为不计分（beta questions），只用于测试，不影响成绩。
  • 因此实际计分题为 90 题。

通过标准

  • 必须答对 63/90（70%） 才算通过。
  • 考试使用电子机考，查阅 MPEP 等参考资料可在系统内完成。

成绩公布

  • 考完后会有一个可选的调查问卷，再之后公布考试成绩。通过会告知 pass 并提供一份非正式成绩（unofficial result），未通过会告知具体分数。
  • USPTO 会在一般 5 个工作日内发正式成绩。

考试通过后

  1. 通过通知
  2. 需要完成的注册材料（Data Sheet + Oath/Affirmation）

注意：

  • 所有材料必须在通知发出后的 2 年内提交，否则需重新考试。
  • USPTO 会进行 moral character 调查。
  • 完成全部流程后，将被注册为：
    • Patent Agent / Design Patent Agent（若无律师资格）
    • 若同时提交州最高法院的律师执照良好证明（6 个月内签发），则登记为 Patent Attorney / Design Patent Attorney。

未通过考试时

  • 会收到你的分数，并可选择预约 Prometric 考后复盘（Review Session）。
  • 复盘时间 2.5 小时，需额外付费给 Prometric。
  • Review session 会提供草稿纸，但不能把纸带出考场。
