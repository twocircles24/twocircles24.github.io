与 XAFS Collaboration 沟通时需要什么样的信息

October 5, 2020

做成文字版方便阅读。

之前已经发成视频 (Bilbili, YouTube)，后来想想觉得还是应该做成文字版当成一个 To-do List 更加合适。

Disclaimer: 该文仅为个人意见，不代表正确答案，请酌情采纳。如果有任何建议，请联系我。

1. 样品状态，以及测量时需要的状态。 Link to this section #

  • 固体：粉末、片状……
  • 液体：悬浊液，溶剂……
  • 气体 （相对少见）

2. 待测的元素及其边 Link to this section #

  • 元素的名称，eg. Pt, Fe…
  • 对应的边，eg. K-edge, L3-edge…

3. 待测元素的浓度 Link to this section #

  • 建议使用质量分数 (wt%)，比如 一个 2 wt% Fe 的铁负载在二氧化硅的载体上。
  • 也有其他的表达方式，比如生物样品就不用 wt%，需要跟其他人沟通好。

4. 样品中其他的组成 Link to this section #

  • 样品中所有的元素及其对应的质量分数，比如 Fe/SiO2 样品中，不仅需要 Fe 的质量分数，还需要 Si 和 O 的质量分数。
  • 这个信息可以直接决定数据质量成败（or 机时能被浪费多少）。

5. 可以提供的样品质量 Link to this section #

  • 明确表明能够给 xxx 克、xxx 毫升的样品。
  • 问清楚对方需要多少样品才可以测量。

6. 对样品的特殊保护/备注 Link to this section #

  • 空气敏感？易潮？
  • 低温装置保护？
  • 有任何危害性？比如易燃易爆、辐射性 etc
  • 容易被同步辐射光源打变质？(aka. beam damage)
  • ……

7. 标准样品的需求 Link to this section #

  • 想知道样品的氧化态，是否需要测量已知价态的不同化合物？
  • ……

8. 实验条件（特别在 in-situ 条件下） Link to this section #

  • 使用何种浓度的气体，什么样的温度，变温速率，气流速率……
  • 保证任何人看到步骤都没有任何的疑问；
  • ……

如果你作为我的 collaboration，实验之前我想从你这里了解更多。 Link to this section #

  1. 前期表征已经得到的信息，包含但不限于价态、元素组成、配位数；
  2. 你想从 XAFS 中获得什么样新的信息，尤其是其他表征手段无法挖掘的信息；
  3. 是否已经对待测样品的结构信息已经有若干个选项。

总结 Link to this section #

  1. 沟通万岁，理解万岁；
  2. 多寄一点样品，尤其是第一次测类似的样品；
  3. 让样品准时到达 collaboration 的手中。
